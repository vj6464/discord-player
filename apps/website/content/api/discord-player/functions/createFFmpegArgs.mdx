### createFFmpegArgs(input, post?): [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)\<[string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)>

Create FFmpeg arguments from an object.

```ts
createFFmpegArgs(
  { i: 'input.mp3', af: ['bass=g=10', 'acompressor'] },
  './out.mp3',
);
// => ['-i', 'input.mp3', '-af', 'bass=g=10,acompressor', './out.mp3']
```

| Parameter | Type                                                                                                                                                                                                                                                                                                     | Optional | Description       |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ----------------- |
| input     | Record\<[string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) \| [ArgPrimitive](/api/types/ArgPrimitive) \| [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)\<[ArgPrimitive](/api/types/ArgPrimitive)>>       | ❌       | The input object. |
| post      | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) \| [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)\<[string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)> | ✅       | N/A               |
