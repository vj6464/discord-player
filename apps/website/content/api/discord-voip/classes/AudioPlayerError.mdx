## AudioPlayerError extends [Error](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error)

An error emitted by an AudioPlayer. Contains an attached resource to aid with
debugging and identifying where the error came from.

```typescript
new AudioPlayerError(error, resource);
```

| Parameter | Type                                                                                            | Optional |
| --------- | ----------------------------------------------------------------------------------------------- | -------- |
| error     | [Error](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error) | ❌       |
| resource  | AudioResource\<unknown>                                                                         | ❌       |

## Properties

### public resource: any

The resource associated with the audio player at the time the error was thrown.
