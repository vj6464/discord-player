## NoSubscriberBehavior

Describes the behavior of the player when an audio packet is played but there are no available
voice connections to play to.

| Property | Type    | Value | Description                                                           |
| -------- | ------- | ----- | --------------------------------------------------------------------- |
| Pause    | 'pause' | N/A   | Pauses playing the stream until a voice connection becomes available. |
| Play     | 'play'  | N/A   | Continues to play through the resource regardless.                    |
| Stop     | 'stop'  | N/A   | The player stops and enters the Idle state.                           |
