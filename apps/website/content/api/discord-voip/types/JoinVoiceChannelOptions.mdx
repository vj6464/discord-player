## JoinVoiceChannelOptions

The options that can be given when joining a voice channel.

| Property  | Type                                                                                                | Value | Description                                             |
| --------- | --------------------------------------------------------------------------------------------------- | ----- | ------------------------------------------------------- |
| channelId | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)   | N/A   | The id of the Discord voice channel to join.            |
| group     | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)   | N/A   | An optional group identifier for the voice connection.  |
| guildId   | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)   | N/A   | The id of the guild that the voice channel belongs to.  |
| selfDeaf  | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | N/A   | Whether to join the channel deafened (defaults to true) |
| selfMute  | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | N/A   | Whether to join the channel muted (defaults to true)    |
