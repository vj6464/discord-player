## VoiceConnectionDisconnectedOtherState

The state that a VoiceConnection will be in when it is not connected to a Discord voice server nor is
it attempting to connect. You can manually attempt to reconnect using VoiceConnection#reconnect.

| Property     | Type                                                                                                       | Value |
| ------------ | ---------------------------------------------------------------------------------------------------------- | ----- |
| adapter      | [DiscordGatewayAdapterImplementerMethods](/api/discord-voip/types/DiscordGatewayAdapterImplementerMethods) | N/A   |
| reason       | AdapterUnavailable \| EndpointRemoved \| Manual                                                            | N/A   |
| status       | Disconnected                                                                                               | N/A   |
| subscription | [PlayerSubscription](/api/discord-voip/classes/PlayerSubscription)                                         | N/A   |
