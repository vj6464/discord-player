## CreateVoiceConnectionOptions

The options that can be given when creating a voice connection.

| Property       | Type                                                                                                | Value | Description                                                                                                     |
| -------------- | --------------------------------------------------------------------------------------------------- | ----- | --------------------------------------------------------------------------------------------------------------- |
| adapterCreator | DiscordGatewayAdapterCreator                                                                        | N/A   | N/A                                                                                                             |
| debug          | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | N/A   | If true, debug messages will be enabled for the voice connection and its related components. Defaults to false. |
