## OggDemuxer extends [Transform](https://nodejs.org/dist/latest/docs/api/stream.html#stream_class_stream_transform)

```typescript
new OggDemuxer(options);
```

| Parameter | Type | Optional | Description                                                |

| options   | \{}  | ✅       | options that you would pass to a regular Transform stream. |
